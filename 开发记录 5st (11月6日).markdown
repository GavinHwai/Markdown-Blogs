### Java 内部类如何被外部类调用

    class Outer{
      private int size=10;
      class Inner{
            public void doStuff(){
                    System.out.println(++size);
            }
      }
      }
      public class Test{
      public static void main(String [] args){
             Outer outer=new Outer();
             Outer.Inner inner=outer.new Inner();
             inner.doStuff();
      }
      }

外部类调用内部类中成员方法（非静态方法）
